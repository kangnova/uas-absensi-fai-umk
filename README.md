# Sistem Absensi UAS FAI UMK

Sistem informasi manajemen absensi panitia dan pengawas UAS Fakultas Agama Islam, Universitas Muhammadiyah Klaten. Dibangun dengan PHP Native + MySQL.

## Fitur Utama
- **Manajemen Data**: Tambah/Edit/Hapus panitia & pengawas (NIP/NIDN unik).
- **Generate QR Code Unik**: Token unik per user, format PNG.
- **Scanner Real-time**: Menggunakan kamera browser (mobile/desktop).
- **Dashboard Admin**: Monitoring real-time, statistik, export Excel/PDF.

## Teknologi
- Backend: PHP 8+ Native, MySQL 8+
- QR Code: endroid/qr-code
- Scanner: html5-qrcode
- Export: PHPExcel/PHPFPDF
